Visualización de distribuciones

Introducción

La forma de visualizar la distribución de una variable depende del tipo de variable:

  • categórica
  • numérica.

Variables categóricas

Una variable categórica

  • Una variable categórica (también denominada variable cualitativa) es aquella cuyas unidades de observación se clasifican en un conjunto de categorías o clases mutuamente excluyentes y colectivamente exhaustivas, las cuales no poseen una magnitud numérica intrínseca por naturaleza.

  • Para examinar la distribución de una variable categórica, puede utilizar un gráfico de barras.

  • La altura de las barras muestra cuántas observaciones ocurrieron con cada valor x.

Carga de librería

Código 
library(tidyverse)
library(palmerpenguins)
Código 
ggplot(penguins, aes(x = species)) + geom_bar()

Ordenar por la frecuencia

En los gráficos de barras de variables categóricas con niveles no ordenados, como la especie de los pingüinos, a menudo es preferible reordenar las barras en función de sus frecuencias.

Para ello es necesario transformar la variable en un factor y luego reordenar los niveles de ese factor.

Ordenar barras por frecuencia

Código 
ggplot(penguins, aes(x=fct_infreq(species))) + geom_bar()

  geom_bar(fill = "steelblue", color = "black") +
  geom_text(

Variables numéricas

Una variable numérica

  • Una variable numérica es aquella que expresa una característica de la unidad de observación mediante un valor escalar perteneciente al conjunto de los números reales (\(\mathbb{R}\)).

  • Las variables numéricas pueden ser continuas o discretas.

  • Una visualización comúnmente utilizada para distribuciones de variables continuas es el histograma.

  • Un histograma divide el eje x en intervalos o segmentos igualmente espaciados y luego utiliza la altura de una barra para mostrar la cantidad de observaciones que caen en cada intervalo.

Histograma

Histograma, código ggplot2

Código 
ggplot(penguins, aes(x = body_mass_g)) + geom_histogram()

Histograma

Podemos controlar el ancho de los intervalos de clase con binwidth

Código 
ggplot(penguins, aes(x = body_mass_g))
  + geom_histogram(binwidth = 200)

Histograma

Calcular el binwidth

Código 
# Número de datos 
n=length(penguins$body_mass_g)
# Número de intervalos 
K=ceiling(1+3.3*log10(n)) 
# Rango 
R=diff(range(penguins$body_mass_g,na.rm = TRUE)) 
# Ancho de los intervalos de clase (binwidth) 
A=R/K
paste("Ancho=",A,"Rango=",R,"Número de int=",K)
[1] "Ancho= 360 Rango= 3600 Número de int= 10"
Código 
ggplot(penguins, aes(x = body_mass_g)) 
  + geom_histogram(binwidth = A)

Histograma

Histograma

Código 
ggplot(penguins, aes(x = body_mass_g)) 
  + geom_histogram(bins=K)

Gráfico de densidad

  • Una visualización alternativa para distribuciones de variables numéricas es un gráfico de densidad.

  • Un gráfico de densidad es una versión suavizada de un histograma y una alternativa práctica, particularmente para datos continuos que provienen de una distribución suave subyacente.

  • Imagínese un histograma hecho con bloques de madera.

  • Luego imagina que dejas caer sobre él un hilo de espagueti cocido.

  • La forma que adoptarán los espaguetis al colocarlos sobre los bloques puede considerarse como la forma de la curva de densidad.

  • Muestra menos detalles que un histograma, pero puede facilitar la comprensión rápida de la forma de la distribución, particularmente con respecto a los modas y la asimetría.

Gráfico de densidad

Código 
ggplot(penguins, aes(x = body_mass_g)) + geom_density()

Dos variables

Visualizando relaciones

  • Para visualizar una relación necesitamos tener al menos dos variables asignadas a la estética de un gráfico.

  • En las siguientes secciones aprenderá acerca de los gráficos comúnmente utilizados para visualizar relaciones entre dos o más variables y las geometrías utilizadas para crearlas.

Una variable numérica y la otra categórica

  • Podemos utilizar diagramas de caja uno al lado del otro.

  • Un diagrama de caja es un tipo de representación visual de las medidas de posición (percentiles) que describen una distribución.

  • También es útil para identificar posibles valores atípicos.

El boxplot consta de …

  • Un cuadro que indica el rango de la mitad central de los datos, una distancia conocida como rango intercuartil (RIC), que se extiende desde el percentil 25 de la distribución hasta el percentil 75.

  • En el medio de la caja hay una línea que muestra la mediana, es decir, el percentil 50, de la distribución.

  • Estas tres líneas le dan una idea de la dispersión de la distribución y de si la distribución es simétrica respecto de la mediana o está sesgada hacia un lado.

  • Puntos que muestran observaciones que caen más de 1.5 veces el RIQ desde cualquier borde de la caja. Estos puntos periféricos son inusuales, por lo que se representan individualmente.

  • Una línea (o bigote) que se extiende desde cada extremo de la caja y va hasta el punto no atípico más lejano de la distribución.

Boxplot de la masa corporal por especie

Echemos un vistazo a la distribución de la masa corporal por especie usando geom_boxplot():

Código

  + geom_boxplot()

Código 
ggplot(penguins,aes(x =body_mass_g,color=species)) + 
  geom_density(linewidth = 0.75) + 
  labs(color="Especies")

  • Podemos asignar species a la estética color y fill (relleno) y usar la estética alpha para agregar transparencia a las curvas de densidad rellenas.

  • Esta estética toma valores entre 0 (completamente transparente) y 1 (completamente opaco).

  • En el siguiente gráfico se establece alpha en 0.5

Código 
ggplot(penguins, aes(x = body_mass_g,
  color = species, fill = species)) +
geom_density(alpha = 0.5)

Diferencia entre ‘map’ mapear y ‘set’ establecer

Tenga en cuenta la terminología que hemos utilizado aquí:

  • Mapeamos (map) variables a la estética si queremos que el atributo visual representado por esa estética varíe en función de los valores de esa variable.

  • De lo contrario, establecemos (set) el valor de una estética.

Gráfico de dispersión

Tres o más variables

  • Podemos incorporar más variables a un gráfico al asignarles una estética adicional.

  • Por ejemplo, en el siguiente diagrama de dispersión los colores de los puntos representan especies y las formas de los puntos representan islas:

Código 
ggplot(penguins,aes(x = flipper_length_mm, y=body_mass_g)) 
  + geom_point(aes(color=species,
                shape=island))

Separar el gráfico en facets

  • Sin embargo, añadir demasiados mapeos estéticos a un gráfico lo vuelve desordenado y difícil de entender.

  • Otra opción, que es particularmente útil para las variables categóricas, es dividir el gráfico en facets, subgráficos que muestran en cada gráfico un subconjunto de los datos.

  • Para dividir el gráfico mediante una sola variable, utilice facet_wrap().

  • El primer argumento de facet_wrap() es una fórmula que se crea con ~ seguido de un nombre de variable.

  • La variable que pase a facet_wrap() debe ser categórica.

facet_wrap()

Código 
ggplot(penguins,aes(x=flipper_length_mm, y=body_mass_g)) 
  + geom_point(aes(color=species,shape=species))
  + facet_wrap(~island)

facet_wrap()

Guardar los gráficos

  • Una vez que hayas creado un gráfico, es posible que quieras guardarlo como una imagen que puedas usar en otra aplicación.

  • Eso se hace con la función ggsave(), que guarda el último gráfico creado en el disco:

Código 
ggplot(penguins, aes(x = flipper_length_mm, y = body_mass_g)) + geom_point()

ggsave(filename = "penguin-plot.png")
